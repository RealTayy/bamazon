# bamazon
bamazon is a CLI that simulates an online store. It contains 3 programs which interact with the same database:
* Customer
    * Can buy products from the store
* Manager
    * View Products for Sale
    * View Low Inventory (Item less then 20)
    * Add to Inventory
    * Add New Product
* Supervisor
    * Can buy products from the store
    * Can buy products from the store
